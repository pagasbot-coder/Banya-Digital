## Цели (напоминание)

| Метрика | Ориентир к неделе 8 |
|---------|---------------------|
| **WAMZ** (North Star) | **≥3 из 4** залов активны за скользящие 7 дней |
| Закрытие смены | **≤30 мин** (было ~45) |

## Ежедневный ритм (кратко)

1. Утро — `/dashboard` (KPI, алерты, чеклисты).
2. Смена — `/finance` при проводках; `/operations` при конфликте кухни (кнопка **Разобрано**).
3. Перед закрытием — чеклист N/M = 100%; FIFO OUT при списании.
